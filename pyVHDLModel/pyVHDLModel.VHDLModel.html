class pyVHDLModel.VHDLModel.ModelEntity[source]¶

ModelEntity is a base class for all classes in the VHDL language model, except for mixin classes (see multiple inheritance) and enumerations.

Each entity in this model has a reference to its parent entity. Therefore a protected variable _parent is available and a readonly property Parent.

Inheritance

Members
